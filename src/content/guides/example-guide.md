# Как добавлять новые гайды

Этот гайд покажет, как создавать новые материалы для сайта.

## Шаг 1: Создайте файл

Создайте новый `.md` файл в папке `src/content/guides/`. Например:

```
src/content/guides/my-new-guide.md
```

## Шаг 2: Добавьте метаданные

В начале файла добавьте блок с метаданными (frontmatter):

```markdown
---
title: "Название вашего гайда"
description: "Краткое описание"
pubDate: 2024-01-28
tags: ["тег1", "тег2"]
---
```

## Шаг 3: Напишите контент

После метаданных пишите обычный Markdown:

- Используйте `#` для заголовков
- Используйте `-` или `*` для списков
- Используйте `` `код` `` для кода
- Используйте `[текст](ссылка)` для ссылок

## Добавление файлов

Чтобы прикрепить файл (Excel, PDF и т.д.):

1. Положите файл в папку `public/files/`
2. В гайде добавьте ссылку: `[Скачать файл](/files/имя-файла.xlsx)`

Пример: [Скачать тестовый файл](/files/example.txt)

## Готово!

После сохранения файла гайд автоматически появится на сайте.
